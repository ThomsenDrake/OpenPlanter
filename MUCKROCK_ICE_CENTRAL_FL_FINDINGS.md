## Executive Summary

A comprehensive search of MuckRock's FOIA database reveals **significant prior FOIA activity related to ICE operations in Florida**, including several completed requests with downloadable documents directly relevant to our Central FL ICE investigation. Two systematic nationwide investigations ("opclaudia" ICE MOUs project and "fxmatt4" local ICE contracts) covered multiple Florida agencies. The most valuable completed requests include:

1. **Polk County Sheriff ICE MOUs/MOAs** — 7 PDF documents spanning 2007–2019 (26 total files)
2. **Broward County Sheriff US Marshals Service Agreement** — completed with PDF
3. **Sarasota County Sheriff ICE agreements** — 3 PDFs including IGSA (2003), MOA (2019), and BOA (2018)
4. **Walton County Sheriff ICE agreements** — MOA + BOA documents
5. **A critical disclosure that Orange County FL jail is run by Orange County Government, NOT the Sheriff's Office** — redirecting the proper FOIA target

### 1. Orange County FL: The Sheriff/Jail Distinction (★★★ HIGH VALUE)

**Request [47521]** — "Sheriffs ICE contracts (Orange County Sheriff's Office)"  
- **Filed:** 2018-01-09 by user fxmatt4  
- **Status:** No responsive documents  
- **Key response (2018-01-17):** *"Please be advised that the Orange County Sheriff's Office and the Orange County Jail are two separate entities, the jail is run by Orange County Government. The Orange County Sheriff's Office has no records responsive. Please contact the jail for any records reference ICE contracts."*  
- **Agency:** Orange County Sheriff's Office FL (MuckRock agency 679, jurisdiction 196)
- **URL:** https://www.muckrock.com/foi/orange-county-196/sheriffs-ice-contracts-orange-county-sheriffs-office-47521/

**Implication:** Any ICE detention/contract requests for Orange County FL must be directed to **Orange County Government** (MuckRock agency 7555), not the Sheriff's Office. This is a critical routing finding for our investigation. No MuckRock user has yet filed an ICE-specific request to Orange County Government.

## Methodology

### API Endpoints Used
- `GET /api_v1/foia/` — FOIA request search (filter by: agency, jurisdiction, tags, title, user, status)
- `GET /api_v1/agency/` — Agency lookup (filter by: jurisdiction, name)
- `GET /api_v1/jurisdiction/` — Jurisdiction lookup (filter by: parent, name)
- `GET /api_v1/project/` — Project metadata
- `GET /api_v1/news/` — MuckRock news articles
- `OPTIONS /api_v1/foia/` — Schema discovery

### Search Strategy
1. **Tag-based search:** ice, immigration, detention, detainer, igsa, opclaudia, 287g
2. **Agency-based search:** All Central FL agencies (OCSO 679, Seminole SO 448, Brevard SO 420, Polk SO 474, Hillsborough SO 775, OPD 447, OC Government 7555)
3. **Jurisdiction-based search:** FL state (34), Orange County FL (196), Seminole County FL (195), Orlando (123), Brevard (182), Polk (204), Hillsborough (382)
4. **User-based search:** fxmatt4 (Frank Matt, ICE contracts investigator), opclaudia/Emma Best (ICE MOUs investigator)
5. **Project-based search:** ICE Insider (233), The New Immigration System (145)
6. **Cross-reference:** FL state-level FOIA requests (962 total) filtered for ICE/immigration keywords

### Limitations
- MuckRock's `search` parameter appears non-functional (returns full unfiltered set)
- `title` filter requires exact match
- No full-text search of request communications/responses via API
- Some requests have embargoed results (not publicly accessible)
- The API does not support OR queries across multiple tags
